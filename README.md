# Dotfiles

[![CircleCI](https://circleci.com/gh/cohei/dotfiles.svg?style=svg)](https://circleci.com/gh/cohei/dotfiles)

## Requirements

- [Nix](https://nixos.org)

## Installation

```shell
curl -L https://dotfiles.cohei.me | sh
```

or

```
wget -O- https://dotfiles.cohei.me | sh
```

## Development

``` shell
docker compose run development
```
